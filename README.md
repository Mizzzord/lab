# Star Tracker

Android приложение для отслеживания звездного неба.

## Возможности

- Отображение звездного неба в реальном времени через камеру
- Отслеживание ориентации устройства с помощью акселерометра, гироскопа и магнитометра
- Отображение звезд и созвездий поверх видеопотока камеры
- Каталог из 200+ звезд с информацией о яркости и координатах

## Требования

- Android 7.0 (API 24) или выше
- Камера
- Акселерометр
- Магнитометр
- Гироскоп

## Установка

1. Клонируйте репозиторий
2. Откройте проект в Android Studio
3. Синхронизируйте Gradle зависимости
4. Запустите приложение на устройстве или эмуляторе

## Использование

1. При первом запуске предоставьте разрешение на использование камеры
2. Наведите устройство на небо
3. Звезды будут отображаться поверх видеопотока камеры
4. Информация об ориентации (азимут, наклон, крен) отображается в левом нижнем углу

## Технологии

- Kotlin
- Android CameraX
- Android Sensor API
- Custom View для рендеринга звезд
